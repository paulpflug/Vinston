Vinston
====

### Install

* Install node.js and mongodb
* Start mongodb
* Install dependencies
```S
npm install -g grunt-cli
npm install -g bower
npm install -g coffee-script
```
* Clone repository
* In repository folder 
```S
npm install 
bower install
```
* Start app
```S
grunt serve
```
* Browse to 
```
localhost:9000
```
